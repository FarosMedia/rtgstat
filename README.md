# TGStat Анализатор

Полнофункциональный инструмент для анализа Telegram каналов и постов с использованием TGStat API.

## 🚀 Возможности

- **Поиск каналов и постов** - Поиск по ключевым словам с фильтрацией по странам
- **Управление проектами** - Создание, редактирование и удаление проектов анализа
- **Система ролей** - Разграничение доступа для разных пользователей
- **Аналитика** - Анализ данных каналов и постов
- **Современный UI** - Интуитивно понятный интерфейс с адаптивным дизайном

## 🛠 Технологии

- **Frontend**: React 18, React Router DOM
- **Стили**: Tailwind CSS
- **Иконки**: Lucide React
- **HTTP клиент**: Axios
- **Графики**: Recharts
- **Работа с датами**: date-fns

## 📦 Установка и запуск

1. **Клонируйте репозиторий**
```bash
git clone <repository-url>
cd tgstat-analyzer
```

2. **Установите зависимости**
```bash
npm install
```

3. **Запустите приложение**
```bash
npm start
```

Приложение будет доступно по адресу: http://localhost:3000

## 📊 API методы

Приложение использует следующие методы TGStat API:

- `searchChannels` - Поиск каналов
- `searchPosts` - Поиск постов
- `getChannelInfo` - Информация о канале
- `getChannelPosts` - Посты канала
- `getChannelStats` - Статистика канала
- `getTopChannels` - Топ каналов
