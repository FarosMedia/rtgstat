# Руководство пользователя TGStat Анализатора

## 🎯 Обзор системы

TGStat Анализатор - это мощный инструмент для анализа Telegram каналов и постов с использованием TGStat API. Система предоставляет интуитивно понятный интерфейс для создания проектов, поиска каналов, анализа данных и управления доступом пользователей.

## 👥 Роли пользователей

### Администратор
- **Полный доступ** ко всем функциям системы
- **Управление пользователями** - создание, редактирование, удаление
- **Управление проектами** - создание, настройка, удаление
- **Аналитика** - доступ ко всем отчетам и данным
- **Настройки системы** - конфигурация API, параметров

### Аналитик
- **Просмотр проектов** - доступ к назначенным проектам
- **Анализ данных** - работа с аналитическими отчетами
- **Поиск каналов и постов** - использование поисковых функций
- **Создание отчетов** - генерация аналитических отчетов

### Менеджер
- **Просмотр проектов** - доступ к назначенным проектам
- **Создание проектов** - создание новых проектов анализа
- **Базовый анализ** - просмотр основных метрик

### Наблюдатель
- **Только чтение** - просмотр данных без возможности редактирования

## 🚀 Начало работы

### 1. Вход в систему
1. Откройте приложение в браузере
2. Система автоматически войдет под учетной записью администратора
3. Для смены пользователя используйте меню профиля в правом верхнем углу

### 2. Навигация
- **Боковая панель** - основная навигация по разделам
- **Сворачивание панели** - кнопка со стрелкой для экономии места
- **Поиск** - глобальный поиск в заголовке
- **Уведомления** - иконка колокольчика для просмотра уведомлений

## 📊 Основные разделы

### Главная панель (Dashboard)
**Назначение**: Обзор всех проектов и ключевых метрик

**Возможности**:
- Просмотр статистики по проектам
- Быстрый доступ к недавним проектам
- Топ каналов по вовлеченности
- Быстрые действия для создания проектов

**Как использовать**:
1. Просматривайте общую статистику в карточках
2. Кликайте на проекты для перехода к детальному анализу
3. Используйте кнопки быстрых действий

### Поиск
**Назначение**: Поиск каналов и постов по ключевым словам

**Возможности**:
- Поиск каналов по названию и описанию
- Поиск постов по содержимому
- Фильтрация по странам
- Просмотр детальной информации о каналах

**Как использовать**:
1. Выберите тип поиска (каналы или посты)
2. Введите поисковый запрос
3. Выберите страну для фильтрации
4. Нажмите "Найти"
5. Просматривайте результаты с возможностью перехода к каналу

### Проекты
**Назначение**: Управление проектами анализа

**Возможности**:
- Создание новых проектов
- Редактирование существующих проектов
- Назначение участников проекта
- Установка ключевых слов для мониторинга
- Управление статусом проекта

**Как создать проект**:
1. Нажмите "Новый проект"
2. Заполните название и описание
3. Выберите статус проекта
4. Добавьте ключевые слова
5. Назначьте участников
6. Сохраните проект

### Аналитика
**Назначение**: Детальный анализ данных каналов и постов

**Возможности**:
- Выбор проекта для анализа
- Настройка временного диапазона
- Просмотр метрик роста подписчиков
- Анализ вовлеченности каналов
- Мониторинг популярных ключевых слов
- Анализ производительности постов

**Как анализировать данные**:
1. Выберите проект из выпадающего списка
2. Установите временной диапазон
3. Просматривайте метрики в карточках
4. Изучайте графики и таблицы
5. Экспортируйте данные при необходимости

### Пользователи
**Назначение**: Управление пользователями и правами доступа

**Возможности**:
- Создание новых пользователей
- Назначение ролей и прав доступа
- Редактирование профилей пользователей
- Удаление пользователей
- Просмотр активности пользователей

**Как управлять пользователями**:
1. Нажмите "Добавить пользователя"
2. Заполните имя и email
3. Выберите роль пользователя
4. Назначьте проекты для доступа
5. Установите пароль
6. Сохраните пользователя

## 🔍 Поиск и фильтрация

### Глобальный поиск
- Доступен в заголовке приложения
- Ищет по проектам, каналам, пользователям
- Результаты отображаются в реальном времени

### Фильтры в разделах
- **По статусу** - активные, приостановленные, завершенные
- **По роли** - администраторы, аналитики, менеджеры
- **По дате** - создание, обновление
- **По названию** - алфавитная сортировка

## 📈 Аналитические метрики

### Основные показатели
- **Общий охват** - суммарное количество подписчиков
- **Активность** - количество постов за период
- **Вовлеченность** - средний процент взаимодействия
- **Количество каналов** - в проекте

### Детальные метрики
- **Рост подписчиков** - динамика по месяцам
- **Топ каналов** - по вовлеченности
- **Популярные ключевые слова** - частота упоминаний
- **Производительность постов** - просмотры, лайки, репосты

## 🔧 Настройки и конфигурация

### Настройки профиля
1. Кликните на аватар в правом верхнем углу
2. Выберите "Профиль" или "Настройки"
3. Измените необходимые параметры

### Настройки проекта
1. Откройте раздел "Проекты"
2. Выберите проект для редактирования
3. Измените параметры и сохраните

### API настройки
- API ключ TGStat настраивается в файле конфигурации
- Для изменения обратитесь к администратору

## 📱 Адаптивность

### Мобильные устройства
- Интерфейс адаптируется под размер экрана
- Боковая панель сворачивается автоматически
- Таблицы становятся прокручиваемыми

### Планшеты
- Оптимальное отображение на средних экранах
- Сохранение всех функций интерфейса

## 🔒 Безопасность

### Управление доступом
- Каждый пользователь имеет свою роль
- Доступ к проектам контролируется администратором
- Пароли хранятся в зашифрованном виде

### Рекомендации
- Регулярно меняйте пароли
- Не передавайте учетные данные третьим лицам
- Выходите из системы при завершении работы

## 🆘 Поддержка и помощь

### Часто задаваемые вопросы

**Q: Как добавить новый канал в проект?**
A: Используйте раздел "Поиск" для поиска канала, затем добавьте его в проект через настройки проекта.

**Q: Почему не отображаются данные в аналитике?**
A: Проверьте, что проект содержит каналы и API ключ корректный.

**Q: Как изменить права доступа пользователя?**
A: В разделе "Пользователи" выберите пользователя и измените его роль.
