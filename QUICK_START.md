# ⚡ Быстрый старт TGStat Анализатора

## 🎯 Минимальные требования

- **Node.js** версии 16+ 
- **npm** (обычно устанавливается с Node.js)
- **Интернет** для установки зависимостей

## 🚀 Запуск за 3 шага

### Шаг 1: Проверка окружения
```bash
# Проверяем Node.js
node --version  # Должно быть v16.0.0 или выше

# Проверяем npm
npm --version   # Должно быть 8.0.0 или выше
```

### Шаг 2: Установка и запуск
```bash
# Устанавливаем зависимости
npm install

# Запускаем приложение
npm start
```

### Шаг 3: Открываем в браузере
Перейдите по адресу: **http://localhost:3000**

## 🎉 Готово!

Ваш TGStat Анализатор запущен и готов к работе!

## 🔧 Если что-то не работает:

### Проблема: "Port 3000 is busy"
```bash
# Используйте другой порт
npm start -- --port 3001
```

### Проблема: "Module not found"
```bash
# Переустановите зависимости
rm -rf node_modules package-lock.json
npm install
```

### Проблема: "API errors"
- Проверьте интернет-соединение
- API ключ уже настроен в коде

Для автоматического развертывания используйте:


Скрипт автоматически:
